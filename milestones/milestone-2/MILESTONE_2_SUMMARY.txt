================================================================================
WHY LSTM FOR BEARING FAULT DIAGNOSIS?
================================================================================

‚úÖ Temporal Pattern Recognition
   - Vibration signals are sequences where order matters
   - LSTM can learn how faults evolve over time

‚úÖ Long-Term Dependencies
   - Cell state remembers important patterns
   - Can capture fault signatures spanning entire signal

‚úÖ Sequential Processing
   - Processes signal step-by-step with context
   - Different perspective than CNN's local windows

‚úÖ Complementary to CNN
   - CNN: "What" features are present (spatial)
   - LSTM: "When" and "how" features evolve (temporal)
   - Best results: Use both approaches!
